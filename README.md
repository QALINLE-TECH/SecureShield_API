# SecureShield API

SecureShield API is a Flask-based backend system implementing modern security practices.

## Features
- User registration & login
- Password hashing (bcrypt)
- JWT authentication
- Role-based access (admin/user)
- Protected routes
- Logout with token blacklist
- Security logging

## Endpoints
- POST /register
- POST /login
- GET /protected
- GET /admin
- POST /logout

## Tech Stack
- Python (Flask)
- bcrypt
- JWT
